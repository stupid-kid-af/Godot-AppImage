# Godot-AppImage
Maintaining godot appimages unofficially
Note:- There is still some work to do for nighlty builds

# Running
  1. Download the lastest AppImage from release section
  2. Make it executable 
      ```bash
      chmod a+x Godot-*.AppImage
      ```
  3. Run the AppImage
      ```bash
      ./Godot-*.AppImage
      ```
